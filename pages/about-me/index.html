About Me

I am a software/data engineer working in the metro DC region. Been doing this open source thing ever since about 2009 when I spent about two years contributing open source code to the Movable Type/OpenMelody ecosystem. I have since repented of ever attempting to to convince people to use Perl and write most of my code in Java or Groovy and occasionally Python.
