NiFi Docker Compose Samples

This project is a collection of Docker Compose configurations for trying out NiFi with different components. It's meant to be a sandbox to help developers get started with various projects. I'm not an expert on any of the components, so support might be limited if something breaks in a new version.

NiFi ArangoDB Bundle

This project is a general NiFi bundle for ArangoDB. I wrote it mainly as a learning exercise for ArangoDB. Users of ArangoDB are welcome to submit issues against it provided that they can explain their setup well enough that it's easy for me to help replicate the issues.

NiFi Email Extraction Bundle

This project is basically a forensics tool that I wrote for NiFi. It provides basic extraction capabilities for MBox, PST, EML and VCard file formats. It's a moving target when it comes to stability on configuring and using it.

NiFi Data Generation Bundle

This project is a bundle that I created for NiFi which provides many ways to generate data. It's mainly meant to provide a way to generate test loads and act as a sort of informal fuzzer for systems that are connected to NiFi. You can use this to generate good data, bad data, combinations of the two and more to test everything from how a systems handle under load, to how well they handle errors.

Avro To Elasticsearch Tool

This project is a Java app that generates opinionated Elasticsearch mappings from Avro source files.
